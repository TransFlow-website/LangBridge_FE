# 번역 작업 화면 설정 안내

## 1. Tiptap 설치 (필수)

프론트엔드 디렉토리에서 다음 명령을 실행하세요:

```bash
cd TransFlow_FE
npm install @tiptap/react @tiptap/starter-kit @tiptap/extension-placeholder
```

## 2. 백엔드 데이터베이스 마이그레이션

백엔드 서버를 재시작하면 `document_lock` 테이블이 자동으로 생성됩니다.

## 3. 테스트 방법

1. 백엔드 서버 실행
2. 프론트엔드 서버 실행
3. "번역 대기 문서" 페이지에서 문서 선택
4. "번역 시작" 또는 "이어하기" 버튼 클릭
5. 번역 작업 화면에서 작업 진행

## 4. 구현된 기능

### 백엔드
- ✅ DocumentLock 엔티티
- ✅ 락 획득/해제 API
- ✅ 락 상태 확인 API
- ✅ 인계 요청 API
- ✅ 번역 완료 API
- ✅ 임시 저장 API

### 프론트엔드
- ✅ 3단 레이아웃 기본 구조
- ✅ Tiptap 에디터 통합 (설치 필요)
- ✅ 락 획득/해제 로직
- ✅ 인계 요청 기능
- ✅ 번역 완료 기능
